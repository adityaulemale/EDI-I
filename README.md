# EDI-I

# An ML- and XAI-Enabled Moderation Framework for Decentralized Social Media

## 📌 Overview
This project proposes a secure and transparent moderation framework for decentralized social media platforms using:

- Machine Learning (ML)
- Explainable AI (XAI)
- Blockchain
- IPFS
- XChaCha20-Poly1305 Encryption

The system detects harmful text and images before they are permanently stored on decentralized platforms.

---

## 🚀 Problem Statement
Decentralized social media platforms face major moderation challenges because blockchain and IPFS storage are immutable.

Once harmful content is uploaded:
- It cannot be easily removed
- It remains permanently accessible
- Traditional moderation systems fail

This project solves this problem using automated AI-based moderation with transparent explanations and secure encryption.

---

## ✨ Key Features

### 🔹 Web3 Authentication
- MetaMask wallet login
- No usernames/passwords required

### 🔹 ML-Based Content Moderation
- BiLSTM model for harmful text detection
- CNN model for harmful image detection

### 🔹 Explainable AI (XAI)
- SHAP explanations for moderation decisions
- Transparent AI predictions

### 🔹 Selective Encryption
- Harmful content encrypted using XChaCha20-Poly1305
- Neutral content stored normally

### 🔹 Decentralized Storage
- IPFS-based storage system
- Tamper-proof file handling

### 🔹 Blockchain Metadata Storage
- Immutable moderation records
- CID, moderation label, encryption status stored on blockchain

## 🧠 System Workflow

1. User connects MetaMask wallet
2. User uploads image/text
3. ML models analyze content
4. SHAP generates explanation
5. Harmful content gets encrypted
6. File uploaded to IPFS
7. Metadata stored on blockchain
8. Content displayed securely
